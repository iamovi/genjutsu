// Features

24

24-Hour Lifespan

Every post, comment, and interaction has a strictly enforced 24-hour timer. True deletes — not just hidden from UI.

Developer First

Optimized for sharing code snippets and technical insights. Post freely without worrying about your permanent record.

Manga Aesthetic

A clean, high-contrast visual style inspired by Japanese manga. Bold. Minimal. Animated with Framer Motion.

Real-time Feed

Instant engagement powered by Supabase. Pure chronological connection. The feed resets every morning.

Secure Login

One-tap OAuth via Google or GitHub. Built on Supabase with Row Level Security. Authenticated effortlessly.

Zero Clutter

No follower counts, no clout chasing, no permanent history. Start every day with a clean slate.

Genjutsu Play

Challenge friends to online multiplayer games — Chess, Tic-Tac-Toe, Rock Paper Scissors, and more. All live, all inside the platform.

Whispers

Ephemeral direct messages that also vanish after 24 hours. Say what you mean. Leave no paper trail.

Dynamic OG Images

Auto-generated open graph images for better social sharing across platforms.

PWA Support

Progressive Web App support for a truly native feel on any device, no APK required.

Community Chat

A built-in space to report bugs, suggest features, talk about the app, or just connect. Admins are always there. Say anything, anytime.
